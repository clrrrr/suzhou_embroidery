v 2.1.1

patch latest 2026-02-26
针对导出质量的两项针对性改进：专业软件兼容性 + 独立查看工具。
新增
  • 新文件 dst_app.py：独立刺绣文件查看器，支持 DST / PES / JEF / VP3 等格式；可命令行传参直接打开
  • DST 查看器支持 STITCH / JUMP / TRIM 三个图层独立显示/隐藏
  • JUMP 以虚线渲染，TRIM 以十字标记渲染，颜色均可自定义
  • 右侧面板：线色列表（可滚动）、文件信息（针迹数/跳针数/剪线数/尺寸）、鼠标坐标实时换算为 mm
优化
  • save_dst() 导出前调用 _optimize_path_order()，对折线进行贪心最近邻排序，同时考虑正向/反向两种走法
  • 基于 scipy.spatial.cKDTree 实现，时间复杂度 O(n log n)，实测 8000 条折线跳针总距离减少约 83%
专业刺绣软件（Wilcom 等）读取的是完整 DST 二进制，包含所有跳针路径。 路径排序优化直接改善了这类软件中的显示混乱问题。

v 2.0.0

major 2026-02-18
核心引擎与交互逻辑全面重构。从"色块填充绣花"转型为"轮廓线段化"工具， 目标场景改为手绘纹样数字化，UI 大幅精简。
架构变更
  • processor.py 完全重写：移除颜色量化、缎面针、长短针、乱针绣等所有填充逻辑
  • 新处理流水线：自适应阈值二值化 → skimage 骨架化 → DFS 折线追踪 → Douglas-Peucker 简化 → DST 导出
  • gui.py 大幅简化：移除所有针法参数控件，仅保留"细节程度"单一滑块（1–100）
新增算法
  • 骨架追踪：8-邻域度数计算（cv2.filter2D），端点优先出发，方向连续性偏好（dot product），交叉点自动断开
  • Douglas-Peucker 简化（cv2.approxPolyDP），epsilon 随细节程度对数映射
  • DST 物理尺寸自动根据图像宽高比计算，最长边固定 200 mm
  • 新增 scikit-image 依赖（骨架化）
移除
  • 颜色量化（_quantize_colors）
  • 缎面针、长短针、乱针绣填充算法
  • 填充角度、行间距、针步长、颜色数等所有 UI 控件
  • 仿射旋转变换（_rotate_mask / _inverse_transform）
骨架化对大面积填充色块仍会产生内部线条（如蕾丝阴影叶片）。 该问题计划在后续版本中通过预处理步骤解决，当前版本暂不处理。

v 1.0.0

initial 2026-02-10
项目初始版本。核心功能：图片 → 颜色量化 → 多种针法填充 → DST 格式导出。
功能
  • 加载任意图片，按颜色量化（中位切割法，2–32 色）自动分层
  • 三种填充针法：缎面针（平行扫描线）、长短针（交错行）、乱针绣（随机方向）
  • 可选轮廓针（cv2.findContours + 等距重采样）
  • 可配置参数：绣面尺寸、针步长、填充行间距、填充角度
  • 针法预览（PIL 渲染，支持缩放/平移）
  • 导出 DST，内置 DST 查看窗口（dst_viewer.py，cv2 渲染 + 超大文件自动抽稀）
  • 处理过程进度回调，后台线程执行，UI 不阻塞
依赖
  • Pillow · numpy · opencv-python · pyembroidery
